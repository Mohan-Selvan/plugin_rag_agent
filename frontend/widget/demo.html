Demo host site

This is a fake host site. The chat bubble in the corner is the entire integration - one script tag added to this page.

Try it

Open the bubble and try a question that needs a knowledge-base lookup, or a small-talk greeting that answers without a tool call. The starter chips inside the widget are pulled from config.yaml.

Embed it on your own site

<script src="https://your-backend.example.com/widget.js" async></script>
